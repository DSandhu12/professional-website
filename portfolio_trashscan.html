üìã Project Overview

TrashScan is an advanced surveillance system developed as part of an academic thesis project by the Agile team. The system leverages computer vision and machine learning to detect and monitor litter in urban environments, helping municipalities maintain cleaner streets and improve waste management efficiency. The solution integrates YOLOv8 object detection with a user-friendly Electron interface to provide real-time monitoring capabilities.

üåü Key Features

  • Real-time Litter Detection: Utilizes YOLOv8 to detect litter objects from live camera feeds or uploaded media
  • Multi-camera Support: Monitor up to 4 cameras simultaneously with flexible viewing layouts
  • Street View Integration: Toggle between standard map view and Google Street View
  • SMS Notifications: Automatically notify street sweepers about high litter areas
  • Data Analytics: Track litter trends by location, time, and density
  • Interactive Dashboard: Visualize litter hotspots on a heat map
  • User Management: Secure login system with role-based access
